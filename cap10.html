Usando links externos

Usado para levar a sites que não são os nossos. Para isso usamos a tag <a>
seguido de href, contendo o link entre aspas. Como no exemplo abaixo:

Link da videoaula

Deixando o href como padrão o usuario arbira o novo link e fechara o seu,
porém adicionando à tag href outros dois parametros (target e rel) o usuario
o novo link vai abrir em uma nova aba do navegador, posibilitando que o usuario
volte para o seu site. Veja os exemplos abaixo:

Usado para levar a links dentro do nosso site. A formação do codigo é igual
aos links externos, porém os links são a localização dos arquivos dentro da
mesma pasta ou URL do anterior. É possivel definir com o rel a sequencia de
