Links que não tem a sua confiança

Existe um rel para links dos quais voce não tem total confiança. o rel="nofollow"
como no exemplo abaixo:

exemplo de nofollow usando o google.com

Para o usuario não muda nada, mas o mecanismo de busca não vai indexar o seu
site com o link que nao tem a sua total confiança. Alguns anunciantes pedem que
não se use o nofollow.

Links para download

Para fazer com que um arquivo seja baixado do seu site é possivel usar <a href>
colocando como referencia a localização do arquivo. Para que navegador enten-
da que é para ser feito o download, dois parametros devem ser adicionados,
download e type="media types que podem ser encontrados no link abaixo."
Veja as imagens demonstrativas:

  • Pagina do download para visualização

  • Link para o download zipado do conteúdo
