Trabalhando com Listas

Listas ordenadas

A tag <ol> é usada para numerar ordenadamente as listas no HTML.
Colocando o padrão TYPE é possivel definir o tipo da lista, enquanto
o START deifine apartir de quando a listagem deve começar.
Como no exemplo abaixo:
Obs o <li> não é mais necessário.

  1. acordar
  2. Ligar para o joão
  3. Tomar café
  4. Escovar os dentes
  5. Ir para o Trabalhando
  6. Almoçar

Listas não ordenadas

Da mesma maneira que a <ol> porém não seguem uma ordem, os
icones não demonstram uma ordem da lista. A tag usada é <ul>

  • Pão
  • Leite
  • Tomate
  • Alface
  • Manteiga
  • Carne

Junção de listas

é possivel criar listas dentro de listas, sendo elas tanto ordenadas quanto
não ordenadas, como no exemplo abaixo:

Meus carros favoritos

  1. Até anos 2000:
    1. Toyota Supra MK4
    2. Ferrari F40
    3. Maclaren MP4/4
  2. Apartir de 2001:
    • Mercedes A45s AMG
    • MAclaren P1
    • Porsche 911 (992)

Lista de definições

Usando a tag <dl> é possivel criar uma lista de definição.
<dt> é usado para o termo a ser definido. <dd> é usado para a definição.
Veja o exemplo abaixo:

HTML:
Linguagem de marcação para a criação de conteudo de um site.
CSS:
Linguagem de marcação para a criação do design de um site.
JavaScript:
Linguagem de programação para a criação de interatividade de um site.
