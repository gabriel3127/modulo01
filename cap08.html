Semantica do HTML 5

O HTML5 se reserva a semantica, o significado das palavras e tags,
deixando a forma ou estilo para a CSS

Algumas tags obsoletas foram descontinuadas do HTML5
Podendo ser encontradas em W3C.org

Formatação de textos

Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (semântica)

Nessa frase temos um termo em destaque usando a tag <strong> (não semântica)

Itálico / Ênfase

Nessa frase, temos um termo em itálico usando a tag <i> (não semantica)

Nesta frase, temos um termo em Ênfase usando a tag <EM> (semantico)

Importante saber que visualmente não a mudança entre Negrito/destaque e Itálico/Ênfase ,porêm para o html existe diferença semantica entre os termos.
Outro observação importante é que as tags <strong> e <EM> são mais novas, enquanto <b> e <i> eram usadas no HTML4 e podem se tornar oboletas.

Formatações adicionais em HTML5

Texto marcado

Podemos criar tambem um texto marcado usando a tag <mark>.
A cor da marcação é feita por css e não html.

para mudar a cor da marcação usar css:

Textos grandes e pequenos

Este é um texto grande, usando a tag <big>, não faz sentido usar letras grandes por isso o W3C desconsiderou a tag <big>.

enquanto isso textos pequenos ainda se mantem relevantes por serem usados em uma serie de
documentos. Para serem feitos usamos a tag <small>

Texto deletado

Podemos marcar um texto como exluído para indicar que ele deve ser lido, mas não considerado.
Para isso usamos a tag <del>.

Texto inserido

Podemos também marcar um texto com inserido para dar uma ênfase e indicar que ele foi adicionado depois. Para isso usasse a tag <ins>.

Existe também o texto sublinhado usando a tag <u>. (não semântica)
Tendo a mesma caracteristica visual de <ins> o <u> sendo não semantica está perdendo o uso.

Texto sobrescrito

Usado para colocar partes do texto para cima, usasse a tag <sup>. Como em x20+3.

Texto subscrito

Usado para colocar partes do texto para baixo, usasse a tag <sub>. Como em H2O.

Citações

A tag <q> é usada quando precisa usar uma citação
Exemplo: Como diria o pai de um amigo: O computador é um burro muito rapido

Pode ser usado também uma <Blockquote>, um pouco maior que a <q>, podendo também
colocar o link para o autor original, usando cite="URL DO ARQUIVO ORIGINAL"
