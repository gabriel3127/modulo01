Capítulo 11 aula 01 - Imagens Dinâmicas

Tente abrir esse site em vários dispositivos diferentes ou simplesmente altere o
tamanho da página no seu navegador. OBS: usando o dev tools é possível mudar
o despositivo de visualização.

a tag <picture> permite que se crie várias fontes para as imagens, podendo o
navegador determinar qual imagem deve aparecer de acordo com o tamanho
da exibição.

Para isso é necessario que se adicione a tag <source> ao picture, fazendo com
que o navegador entenda que se o tamanho da tela for menor do que o determinado
pelo media o proprio navegador vai exibir a outra imagem.
